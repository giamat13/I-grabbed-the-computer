# 🖱️ Mouse Locker (Batch & PowerShell)

כלי פשוט ומבוסס סקריפט המאפשר "לנעול" את סמן העכבר לפינה השמאלית העליונה של המסך עד להזנת סיסמה נכונה. 
מתאים למתיחות, הדגמות טכניות או הגנה בסיסית על המחשב.

---

## 🚀 תכונות עיקריות
- **הגדרה דינמית:** הגדרת שם משתמש וסיסמה בכל הפעלה מחדש.
- **נעילת עכבר אקטיבית:** שימוש ב-PowerShell כדי למשוך את הסמן לנקודה (0,0) בלולאה מהירה.
- **התראות מערכת:** הקפצת חלונית הודעה (Message Box) ברגע הנעילה.
- **מנגנון אימות:** דרישת סיסמה לצורך שחרור המשאבים וסגירת התהליך.

## 🛠️ הוראות שימוש
1. **יצירת הקובץ:** העתק את הקוד לקובץ טקסט ושמור אותו בשם `Lock.bat`.
2. **הפעלה:** הרץ את הקובץ. תתבקש להזין שם וסיסמה.
3. **הנעילה:** לאחר האישור, העכבר יינעל לפינה השמאלית.
4. **שחרור:** הקלד את הסיסמה בחלון ה-CMD ולחץ Enter.

## ⚠️ הערות ואבטחה
- **למטרות לימודיות בלבד:** אין להשתמש בסקריפט זה למטרות זדוניות.
- **עקיפה:** ניתן לעקוף את הנעילה באמצעות המקלדת (Ctrl+Alt+Del או Alt+F4 על החלון).
- **דרישות:** מערכת הפעלה Windows עם הרשאות להרצת PowerShell (מוגדר כברירת מחדל ברוב המערכות).
